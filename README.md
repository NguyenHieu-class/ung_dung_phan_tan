## Hướng dẫn Cài đặt

1. **Cài đặt Python 3.x**: Đảm bảo bạn đã cài đặt Python 3.x.
2. **Cài đặt các thư viện cần thiết**:
    ```bash
    pip install -r requirements.txt
    ```
3. **Chạy ứng dụng**:
    - Để chạy bài tập:
      ```bash
      python baitap/tenbaitap.py
      ```
